The M3X XM-L2 Triton is a high-output long-throw tactical flashlight with a tailcap switch and a family of optional tactical accessories. The M3X XM-L2 features two brightness levels and a strobe mode, easily set by the mode selector built into the head. The tailcap switch allows momentary activation as well as clicking on and off.
